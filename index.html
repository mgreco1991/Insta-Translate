Insta-Translate!

To use this app, begin by selecting the language you would like to translate from and the language you would like to translate to from the dropdown boxes. Then, simply type what you would like to be translated into the text input bar, and click the "Translate!" button. You will see the translated text displayed in the white box on the lower portion of the screen. This application sends a GET request to the Yandex® API for language translation services. In turn, the server responds by displaying the translated text on the client's DOM.
